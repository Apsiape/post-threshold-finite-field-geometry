## Threshold theorem note

**Status: PROVED.**

The frozen threshold statement is:

For every odd prime `p` and every integer `k >= 2`, the full ambient additive value equals `1/3` if and only if `k <= 3p/(p-1)`.

Two immediate arithmetic consequences matter operationally:

- for `p = 3`, the threshold cutoff is `k <= 4`;
- for every odd prime `p >= 5`, the threshold cutoff is `k <= 3`.

This theorem is part of the public release package and is represented in `src/post_threshold_geometry/formulas.py` and in the verification scripts.
